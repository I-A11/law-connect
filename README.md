## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Then, Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

If you fork it you will need to install node_modules:

```bash
npm install
#or
yarn install
```
